FreeCodeCamp API Basejump: URL Shortener Microservice

Get ShortURL

    User stories:

  • I can pass a URL as a parameter and I will receive a shortened URL in the JSON response.
  • When I visit that shortened URL, it will redirect me to my original link.

Example creation usage:

https://salty-quilt.glitch.me/new/https://www.google.com
https://salty-quilt.glitch.me/new/http://foo.com:80

Example creation usage:

{"original_url": "https://www.google.com", "short_url": "5492", "_id": "59acec14fed4f63b7b599627"}

Usage:

https://salty-quilt.glitch.me/5492

Will redirect to:

https://www.google.com/
